## 吉祥物：Bolot（定稿）

| 项 | 说明 |
|----|------|
| 名称 | **Bolot** |
| 形象 | 圆滚河豚 / 气球鱼（puffer / balloon fish）——鼓鼓的身体、小鳍、温和表情 |
| 气质 | 轻量、可膨胀的上下文、不刺人（对照「compact / context」产品隐喻，非抄袭） |
| 终端 | 欢迎区可用 **5–8 行** 原创 ASCII；`NO_COLOR` / plain 时不强制图形 |
| 禁止 | 不使用其它产品的官方 logo、吉祥物轮廓或商标字样 |

未采用候选：Nyxkit / Pipkin / Glim（保留命名空间，不启用）。

## 文案语气

- 短、清楚、可操作；命令与错误信息英文为主（与现有 CLI 一致）。
- 不出现竞品品牌名（文档对照说明除外）。
